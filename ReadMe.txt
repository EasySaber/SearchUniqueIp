Задание:

Дан простой текстовый файл с IPv4 адресами. Одна строка – один адрес, примерно так:
```
145.67.23.4
8.34.5.23
89.54.3.124
89.54.3.124
3.45.71.5
...
```
Файл в размере не ограничен и может занимать десятки и сотни гигабайт.
Необходимо посчитать количество уникальных адресов в этом файле, затратив как можно меньше памяти и времени.

Решение:
-Построчное чтение файла, с добавлением в таблицу SQLite
-Примерная скорость обработки - 1млн.IP/8сек, зависит от типа накопителя SSD/HDD
-Требуемое место на жеством диске 1млн.IP/53,2мб
-После обработки создается БД UniqueIp.db, с уникальными IP из файла-обработки
